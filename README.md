## Git Commands Explained
- `git status` → Shows the current state of your working directory (changes staged, unstaged, or untracked).
- `git log` → Displays commit history.
- `git branch` → Lists all branches in the repository.
- `git add .` → Stages all changes in the current folder.
- `git commit -m "message"` → Saves changes with a descriptive message.
- `git push` → Uploads commits to GitHub.
